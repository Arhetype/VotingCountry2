# Голосовалка по странам (локально)

Простой статический проект: список стран, кнопка голосования, проценты и подсветка на карте. Хранение — `localStorage`, БД не требуется.

## Как запустить

Откройте файл `index.html` двойным кликом или в любом статическом сервере.

- macOS: щёлкните `index.html` — откроется в браузере.
- Либо из терминала: `python3 -m http.server` и перейдите на `http://localhost:8000`.

## Как это работает

- Файл `app.js` хранит голоса в `localStorage` по ключу `country_votes_v1`.
- Список стран и их ISO A3 коды находятся прямо в `app.js` и синхронизированы с `assets/world_simple.svg` (id элементов SVG равны кодам стран).
- Максимум голосов подсвечивается зелёным, минимум — красным, страны без голосов — серым. В таблице выводятся абсолютные значения и проценты.

## Расширение до полной карты мира

Сейчас в `assets/world_simple.svg` упрощённая карта с несколькими крупными странами (для демонстрации). Чтобы покрыть весь мир:

1. Замените файл `assets/world_simple.svg` на более детальную карту, где id каждого региона — ISO A3 код страны.
2. Дополните массив `COUNTRIES` в `app.js` нужными странами с соответствующими кодами.
3. Логика подсветки и процентов работать будет без изменений.

Подходит любой SVG, в котором каждую страну можно идентифицировать по `id` (например `USA`, `RUS`, `CHN`, и т.д.).

## Сброс данных

Кнопка «Сброс» очищает счетчики (но не сам ключ в localStorage).

## Замечания

- При первом открытии все значения нулевые, поэтому подсветка max/min не применяется до появления первого голоса.
- Проценты считаются от общей суммы голосов, округление — до одного знака после запятой.
